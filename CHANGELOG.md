### Features

- 最轻量的Minimal API
- 统一的接口响应格式返回
- 自定义异常处理和数据持久化
- 自定义请求处理中间件
- 参数签名中间件
- 内置健康检查路由
- 基于System.Text.Json的序列化和反序列化
- 支持Sqlite、MySql、SqlServer等主流数据库
- Swagger接口文档
- 参数自动验证
- 认证和授权
- 授权策略支持
- 限流策略支持
- 跨域策略支持
